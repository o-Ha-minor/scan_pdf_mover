# scan_pdf_mover

## 概要
共有スキャナフォルダに保存されたPDFを、指定フォルダへ自動移動し、
日付付きファイル名へリネームする業務改善ツールです。

## 作成背景
スキャン後のPDFを手作業でリネーム・分類していたため、
ファイル名ミスや移動漏れが発生しやすい課題がありました。
本ツールでは、対象PDF名・保存名・移動先をGUIで指定し、
複数ファイルを一括処理できるようにしました。

## 主な機能
- ScanShareフォルダから指定PDFを検索
- 日付付きファイル名へ変換
- 指定フォルダへ移動
- 複数ファイルの一括処理
- 同名ファイルがある場合の連番付与
- 処理ログ出力
